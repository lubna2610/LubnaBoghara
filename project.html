Food@U

- food ordering system

This project intends to create an web appliacation that provide online food ordering system. This website is very easy to use and user interactive features provides more look and feel. Food@u website holds many cities and provide delivery in those cities . All the cities contain many zones from which user can easily select their fav. zone and order the food. Users can create an account or can login in this website.
